# bash

Make Your Directory Rom

For Example (arrow-10.0) :

```bash
    mkdir arrow-10.0
```

Clone This Repo


```bash
    git clone https://github.com/kambium30/bash.git -b master bash-sh
```

Run Bash

```bash
    bash bash-sh/arrow-10.0.sh
```

If you want to edit

```bash
    nano bash-sh/arrow-10.0.sh
```
