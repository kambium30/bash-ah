# bash

Make Your Directory Rom

For Example (carbon) :

```bash
    mkdir carbon
```

Clone This Repo


```bash
    git clone https://github.com/kambium30/bash.git -b master bash-sh
```

Run Bash

```bash
    bash bash-sh/carbon.sh
```

If you want to edit

```bash
    nano bash-sh/carbon.sh
```
